# Contributing

Contributions should improve evidence quality, point-in-time correctness,
accounting, testing, data-health reporting or accessibility.

Before opening a pull request:

1. run `python3 scripts/check_public_release.py`;
2. run the relevant tests;
3. state which data date and accounting convention were used;
4. distinguish observations from investment conclusions;
5. never include tokens, paid datasets, personal portfolio records or broker data.

Primary evidence may support, caution or contradict a thesis. Automated keyword
or financial-threshold hits must remain review candidates until a human checks
the source and records a dated conclusion.
